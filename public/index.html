FAQ

Q. What is OptimusCP?

A. OptimusCP is a cloud server control panel which can be used to manage the currently supported linux distros like Ubuntu & Debian. Unlike other control panels OptimusCP doesn't utilize any of your server resources as it doesn't install anything remotely.

Q. How secure is this?

A. The accessibility of your server is restricted to you and only OptimusCP, none of our agents have access to your servers.

Q. How does this work?

A. OptimusCP act as a middle man who automates your tasks which are tedious and connects to your server directly to perform these. OptimusCP uses PAM for securing the connection plus the server can't be accessed by any one other than OptimusCP and you.

Q. Did we miss something ?
